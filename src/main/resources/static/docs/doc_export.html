Share

Share your project.
GeneWarrior will save your project on its server and generate a link. You can use this link to share your project with your collaborators.

Be aware that your data will be publicly visible for everyone having the link.

Export

Export your sequences for reuse.

Save Sequences as FASTA
DNA and protein sequences are exported as multiFASTA file. Other entries (such as Alignments) are not exported.

Save Alignments as CLUSTAL
DNA and protein alignments are exported in the CLUSTAL format. Other entries (such as sequences) are not exported. More information about the CLUSTAL format can be found on the BioPerl wiki.

Save as static HTML
Export your project as a single HTML file. This is the best way to keep your formatting and import into Microsoft Word, Powerpoint etc.

Print
Send entries to printer.
