Importing Sequences

DNA and Protein sequences can be imported to your project.
Enter them either one by one into the text field and clicking the "Add" button, or enter them as multiFASTA format into the text field.
DNA or protein sequences are automatically distinguished.

MultiFASTA format
The FASTA format starts with the first line describing the sequence name starting with a 'greater than' (>) symbol, followed by the actual DNA or protein sequence
Example:
>Sequence1
AGATATTTTTGTAGGAAATCATGTCATGTTTGGACCAAATGTTACGATTG
ACACGGCGACACATCCTGTCTCGCTTGACTTACGTAAACGAGGGGCTCAA
TATAATAAAAAAGTTTATATTGAAGAAAATGTTTGGCTAGGTGCTGGTGT
AATTGTGTTACCTGGTGTTAGAATTGGGAAAAATAGTG
