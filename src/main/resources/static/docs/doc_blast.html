BLAST

Send your selected sequences to NCBI's BLAST website.

Instructions

Select one or more sequences of one type (DNA or protein) and click on "NCBI BLAST for DNA", or "NCBI BLAST for Proteins", respectively.
NCBI's BLAST website opens up in a new tab, having the selected sequences already filled in. Refer to the NCBI'S BLAST documentation for further information.
