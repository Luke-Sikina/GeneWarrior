This page is part of the GeneWarrior Documentation. Go to the main site of GeneWarrior

GeneWarrior is an online sequence manipulation tool to perform common tasks such as managing sequences, calculating the reverse complement of a sequence, translating nucleotide to protein sequences, creating alignments, sequence logos and phylogenetic trees.

Sequence display is performed client-side using HTML5 and JavaScript, calculations are performed server side using Java Servlet, MUSCLE software for alignments and WebLogo for sequence logos.
