Reverse complement

Create the reverse complement, the complement or the reverse sequence of a DNA sequence.

See What is a reverse complement sequence? and How to create a reverse complement sequence.

This command is only valid for DNA sequences.

List of all complement nucleotides:
Base Name / Bases Complementary Base
A Adenine T
C Cytidine G
G Guanidine C
T Thymidine A
Y Pyrimidine (CT) R
R Purine (AG) Y
S Strong (GC) S
W Weak (AT) W
K Keto (TG) M
M Amino (AC) K
B not A (CGT) V
D not C (AGT) H
H not G (ACT) D
V not T (ACG) B
N Unknown (ACGT) N
Uridine (U) is automatically converted to Thymidine (T) upon entry.
